--------------------------------------------------------------------------------
- 1 - Compiling EOS-SKL
--------------------------------------------------------------------------------
    
    From a terminal prompt, simply type:
        
        make all
        
    The EOS-SKL should be build, as well as test files.
    
--------------------------------------------------------------------------------
- 2 - Testing EOS-SKL
--------------------------------------------------------------------------------
    
    Once EOS-SKL is compiled, you can run a test file by using the following
    command:
        
        make test
        
    This will run the EOS-SKL test file, displaying system informations.
    
--------------------------------------------------------------------------------
- 3 - Installing EOS-SKL
--------------------------------------------------------------------------------
    
    The EOS-SKL can be installed by using the following command:
        
        make install
        
--------------------------------------------------------------------------------
- 4 - Un-installing EOS-SKL
--------------------------------------------------------------------------------
    
    The EOS-SKL can be installed by using the following command:
        
        make uninstall
